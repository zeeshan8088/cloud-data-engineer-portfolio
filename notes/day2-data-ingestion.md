# Day 2 – Data Ingestion & Raw Layer

What I did:
- Created a raw dataset in BigQuery
- Simulated file-based ingestion using CSV
- Loaded raw data into BigQuery tables
- Ran aggregation queries on ingested data

What I understood:
- Data pipelines usually start with files (CSV/JSON)
- Raw data should be stored separately
- BigQuery is often the analytics layer, not the landing zone

Why this matters:
Separating raw and processed data is a core data engineering best practice.
